# ADF_website

## Clone repo
```
git clone --recursive https://github.com/nyanfox87/ADF_website.git
```

## Start Backend
```
cd ADF_website
docker compose up
```
Error: .hf_token not found. To fix this, create a file at tts/index-tts/.hf_token containing your Hugging Face token.


## Start Frontend
```
cd frontend
npm install
npm run dev
```
